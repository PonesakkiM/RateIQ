# RateIQ – Frontend Assets

Place logo files here:

| File | Used when |
|---|---|
| `logo_dark.png` | Dark mode sidebar + header |
| `logo_light.png` | Light mode sidebar + header |

Both files are optional. If missing, the text-based logo renders as fallback.

## Logo specs (from brand guide)
- Dark mode: Purple/indigo "R" icon with dot accent on dark `#0A0F1E` background
- Light mode: Purple "R" icon with blue dot accent on white/light background
- Recommended size: 256×256 px (square, transparent or solid bg)
